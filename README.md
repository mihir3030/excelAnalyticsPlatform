## Tech Stack

**Frontend:**
- React.js (Vite)
- Chart.js / D3.js / Three.js (for 3D)
- Tailwind CSS
- React Dropzone (for file uploads)

**Backend:**
- Node.js
- Express.js
- Mongoose (MongoDB ODM)
- Multer (file handling)
- XLSX (Excel parsing)
